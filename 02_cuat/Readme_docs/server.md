### Compás

Muestra el el rumbo o el ángulo que hay entre el norte y el eje X del sensor. Si la medición de aceleración en el eje Z es menor a 9m/s<sup>2</sup> (caso de la figura) la medición es erronea y se muestra la advertencia en pantalla.
